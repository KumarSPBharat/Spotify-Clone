# Spotify Clone Frontend

A basic Spotify-inspired music player frontend built using **HTML** and **CSS**.  
This project recreates the visual layout of the Spotify Web Player, including a sidebar, music cards, sticky navigation bar, and bottom music player.

> This is a frontend UI clone made for learning purposes. It is not an official Spotify product.

## Project Description

This project is a static frontend clone of Spotify's web player interface. It focuses on layout design, styling, and UI structure using core web technologies. The project includes a left navigation sidebar, library section, playlist cards, featured chart sections, and a bottom music control bar.

It is suitable for beginners who want to practice HTML structure, CSS Flexbox, responsive design basics, icons, fonts, and modern UI styling.

## Features

- Spotify-inspired dark theme UI
- Sidebar with Home, Search, and Library sections
- Sticky top navigation bar
- Recently played section
- Trending music cards
- Featured charts section
- Bottom music player layout
- Album image and song information area
- Player control icons
- Volume control UI
- Responsive hiding of some navbar options on smaller screens

## Tech Stack

- **HTML5**
- **CSS3**
- **Font Awesome Icons**
- **Google Fonts - Montserrat**

## How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/your-username/spotify-clone-frontend.git
```

2. Open the project folder:

```bash
cd spotify-clone-frontend
```

3. Open `index.html` in your browser.

You can also use the **Live Server** extension in VS Code for a better development experience.

Local system paths will not work after uploading the project to GitHub or deploying it online.
